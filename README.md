# IT Intern Portfolio Website

A clean, professional, and modern portfolio website designed for IT interns completing their OJT (On-the-Job Training).

## Features

- **Hero Section**: Introduction with name and current role
- **About Me**: Showcase your passion and career goals
- **Skills Section**: Display both technical and soft skills
- **OJT Projects/Task Log**: Document your learning journey
- **Contact Form**: Allow visitors to reach out
- **Social Links**: Connect via LinkedIn, GitHub, and Email
- **Responsive Design**: Works on all devices

## Color Palette (Professional IT Industry)

- **Primary**: #2C3E50 (Dark Blue-Gray) - Professional, trustworthy
- **Secondary**: #3498DB (Bright Blue) - Tech-forward, modern
- **Accent**: #1ABC9C (Teal) - Fresh, innovative
- **Dark**: #1A252F (Deep Navy) - Depth, sophistication
- **Light**: #ECF0F1 (Light Gray) - Clean, minimal

## Customization Guide

### 1. Personal Information
Edit `index.html` and replace:
- `[Your Name]` with your actual name
- `[Cybersecurity/Web Development/Network Administration]` with your focus area
- Update the hero description and about section

### 2. Skills
Modify the skills section in `index.html`:
- Add/remove technical skills
- Adjust skill bar percentages (in `style` attribute)
- Update soft skills tags

### 3. OJT Projects
Update the project cards with your actual tasks:
- Change dates, titles, and descriptions
- Modify icons (Font Awesome classes)
- Add/remove project cards as needed

### 4. Social Links
Replace `#` in the contact section with your actual links:
- LinkedIn profile URL
- GitHub profile URL
- Email address (use `mailto:your.email@example.com`)

### 5. Contact Form
The form currently shows an alert. To make it functional:
- Integrate with a backend service (e.g., Formspree, EmailJS)
- Or use a serverless function (e.g., Netlify Forms, Vercel)

## How to Use

1. Open `index.html` in a web browser to view locally
2. Customize the content as described above
3. Deploy to a hosting service:
   - GitHub Pages (free)
   - Netlify (free)
   - Vercel (free)
